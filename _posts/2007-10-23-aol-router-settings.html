I don't use AOL (Thank the Lord) but if I did I think I'd cry.

I've just had to set up an AOL router (for the third time) due to AOL's particular router settings.  So this post is for my benefit (for the next time I'll have to do it) and for whoever else is looking for the settings.  They may also be applicable to Linux Boxes trying to connect without a router, so here you go.

  1. Make sure the encapsulation is set to PPPoA, not PPPoE.

  2. Select VC with QOS type UBR.

  3. Disable the auto-detect option for VC settings.

  4. Manually input VPI value 0 and VCI value 38.

  5. Enter your username (user@aol.com) and password.

I hope this is as useful for you as it has been for me.
